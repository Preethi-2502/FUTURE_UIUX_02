# Art Learning App – UI/UX Design Project

## Project Overview

Art Learning App is a mobile application UI/UX design that helps users discover art courses, connect with professional tutors, and book personalized learning sessions. The application is designed to provide a simple, engaging, and creative learning experience for art enthusiasts of all skill levels.

## Objective

The objective of this project is to create an intuitive and visually appealing platform where users can explore art courses, interact with tutors, and schedule learning sessions seamlessly.

## Key Features

* Browse art courses and learning programs
* Create and share artwork/projects
* Explore professional tutor profiles
* Book one-on-one learning sessions
* View booking confirmations
* Manage personal profile and learning activities

## Screens Designed

### 1. Home Page

* Displays featured courses and recommendations
* Provides quick access to learning resources and navigation

### 2. Courses Page

* Lists available art courses
* Allows users to explore different learning categories

### 3. Create Screen

* Enables users to create and showcase their artwork or projects
* Encourages creativity and engagement within the platform

### 4. Tutors Page

* Displays available tutors with brief information
* Helps users find suitable instructors based on their interests

### 5. Tutor Profile Page

* Shows detailed tutor information, experience, ratings, and expertise
* Provides an option to book a learning session

### 6. Booking a Session Page

* Allows users to select a date and time for a session
* Simplifies the appointment booking process

### 7. Booking Confirmed Page

* Confirms successful session booking
* Displays booking details and next steps

### 8. Profile Page

* Displays user information and activity
* Allows users to manage their account details

## Design Process

### Research

Conducted research on educational and creative learning applications to understand user needs and expectations.

### Wireframing

Created structured layouts to define navigation, screen hierarchy, and user flow.

### UI Design

Designed visually appealing interfaces using modern typography, creative illustrations, and consistent color schemes.

### Prototyping

Developed interactive designs demonstrating the complete booking journey from course exploration to session confirmation.

## Design Principles

* User-Centered Design
* Consistency
* Accessibility
* Simplicity
* Clear Navigation

## Tools Used

* Figma
* Canva
* Adobe Illustrator (Optional)

## Target Audience

* Art students
* Hobby artists
* Beginners interested in creative learning
* Individuals seeking professional art guidance

## Conclusion

The Art Learning App provides a seamless and engaging platform for discovering art courses, connecting with tutors, and booking personalized learning sessions. The design focuses on usability, creativity, and an enjoyable learning experience.
